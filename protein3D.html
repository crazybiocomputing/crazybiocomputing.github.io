1-1- wwPDB

The main website of the Protein Data Bank (PDB) is wwPDB and contains all the documentation about the validation and deposition of an atomic 3D structure.

In this documentation, three different file formats are used in PDB as detailed here.

  1. PDBx/mmCIF
  2. PDBML/XML
  3. PDB

Among them, we are only interested in the PDB file format.
