Introduction

CrazyBiocomputing est un site web contenant différents mini-jeux dans les domaines de la bioinformatique et de l'analyse d'images. Ces mini-jeux constituent une aide aux étudiants de biologie pour apprendre à utiliser les outils informatiques dédiés à la biologie.

Les jeux sont regroupés en deux categories : l'une pour la bioinformatique (onglet Bioinformatics) et l'autre pour l'analyse et le traitement des images scientifiques (onglet Image Processing).

Dans la rubrique Docs sont rassemblés des supports de cours pouvant aider à la résolution des enigmes.

What do I need

  • A web browser: Firefox
  • An office suite:OpenOffice.org
  • A 3D protein viewer like OpenRasmol (or equivalent)
  • Bioinformatics search engines: SRS and/or Entrez
  • Proteomic tools: Expasy
